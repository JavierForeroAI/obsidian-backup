## Guía rápida de variables comunes

| Variable | Qué es |
|----------|--------|
| `<NOMBRE>` | Nombre del paciente o persona |
| `<CIUDAD>` | Bogotá / Medellín / Barranquilla / Bucaramanga / Panamá |
| `<TECNICA>` | FUE o DHI |
| `<INJERTOS>` | Cantidad numérica, ej. "2.500" |
| `<MEDICO>` | Dr. Fabián Carreño u otro médico |
| `<ASESOR>` | Nombre del asesor de seguimiento |
| `<FECHA_CIRUGIA>` | Fecha en formato largo, ej. "viernes 23 de mayo" |
| `<SEMANA>` | Rango de fechas, ej. "12-18 mayo 2026" |
| `<KEYWORD_PRINCIPAL>` | Término exacto para SEO, ej. "implante capilar Bogotá precio" |
