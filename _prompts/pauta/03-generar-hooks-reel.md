# Generador de Hooks para Reel

## Cuándo usar
Cuando necesitás ideas de apertura para Reels de Instagram o TikTok sobre implante capilar.

## Variables que tenés que reemplazar
- `<TEMA>`: ej. "miedo al dolor", "precio", "naturalidad del resultado", "recuperación"
- `<DURACION>`: ej. "30 segundos", "60 segundos"
- `<AUDIENCIA>`: ej. "hombres 30-45, alopecia androgenética grado III-V", "mujeres con alopecia post-parto"

## Prompt
Generá 7 hooks creativos para Reel de Instagram sobre el tema: <TEMA>

Contexto:
- Clínica: Innovart Medical IPS — implante capilar FUE/DHI en Colombia y Panamá
- Ticket: $6M-$8M COP
- Diferenciador: precio 50% menor que competencia (Mediarte: $14M-$17.5M)
- Audiencia: <AUDIENCIA>
- Duración del reel: <DURACION>

Reglas de compliance:
- NO usar: "garantizado", "permanente para siempre", "sin riesgos", "100% efectivo"
- SÍ usar: "resultados naturales", "técnica avanzada", "alta satisfacción"
- Siempre hay un disclaimer de "resultados varían" disponible para el caption

Para cada hook, dame:
1. El texto exacto del hook (máx 10 segundos de speech)
2. Emoción que activa (curiosidad / miedo / sorpresa / empatía / aspiración)
3. Visual sugerido para acompañar
4. Por qué funciona para esta audiencia (1 línea)

Ordenados de más a menos potente según tu criterio.

## Output esperado
7 hooks en tabla con texto, emoción, visual y razonamiento. Listos para pasarle al editor o al cliente.

## Ejemplo de uso real
"Generá 7 hooks para reel sobre el miedo al dolor del implante, para hombres 35-50, reel de 45 segundos"
