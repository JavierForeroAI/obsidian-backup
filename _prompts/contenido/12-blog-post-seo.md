# Blog Post SEO para Clínica Capilar

## Cuándo usar
Para generar artículos de blog optimizados para SEO en el sitio de Innovart Medical (Shopify).

## Variables que tenés que reemplazar
- `<KEYWORD_PRINCIPAL>`: ej. "implante capilar Bogotá precio"
- `<KEYWORD_SECUNDARIA>`: ej. "costo trasplante capilar Colombia", "FUE vs DHI"
- `<INTENCION>`: ej. "informacional" (quiero saber) / "transaccional" (quiero comprar) / "comparación"
- `<CIUDAD>`: ej. "Bogotá" (para SEO local)

## Prompt
Escribí un artículo de blog SEO-optimizado para Innovart Medical IPS sobre la keyword: <KEYWORD_PRINCIPAL>

Parámetros SEO:
- Keyword principal: <KEYWORD_PRINCIPAL>
- Keyword secundaria: <KEYWORD_SECUNDARIA>
- Intención de búsqueda: <INTENCION>
- Ciudad: <CIUDAD>
- Longitud objetivo: 1.200-1.800 palabras
- Nivel de lenguaje: profesional pero accesible (no jerga médica sin explicar)

Estructura obligatoria:
1. Título H1 con keyword (máx 60 caracteres)
2. Meta description (máx 155 caracteres, con CTA)
3. Introducción (150 palabras, incluir keyword en primeras 100 palabras)
4. 4-5 secciones H2 con subtemas relevantes
5. Al menos 1 tabla comparativa o lista con viñetas
6. Sección "¿Por qué Innovart?" con diferenciadores reales
7. CTA final: valoración gratuita en <CIUDAD>
8. FAQ: 3 preguntas frecuentes con respuestas (schema FAQ)

Compliance:
- No usar claims médicos no respaldados
- Si se mencionan resultados: agregar "(los resultados varían según cada paciente)"
- No comparar precios de competidores por nombre en el artículo
- Sí mencionar el rango de precio de Innovart: "$6M-$8M COP"

Al final del artículo, dame:
- Sugerencia de URL slug
- Suggested internal links (3 páginas del sitio a las que enlazar)
- External link sugerido (autoridad médica, ej. artículo académico sobre FUE)

## Output esperado
Artículo completo listo para publicar, con metadata SEO, estructura H1/H2, FAQ y notas de publicación.

## Ejemplo de uso real
"Escribí un artículo sobre 'implante capilar Bogotá precio' con intención transaccional para posicionar en Bogotá"
