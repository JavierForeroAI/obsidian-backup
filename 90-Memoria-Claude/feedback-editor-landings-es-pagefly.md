# Editor de landings = PageFly (NO GemPages)

Las landings de ciudad (`/pages/implante-capilar-[ciudad]`) se editan **siempre en PageFly**.

**Por qué importa:** En el menú de Apps de Shopify aparecen tanto **GemPages Builder** como **qikify Form Builder**, lo que induce a error. El editor visual correcto para estas páginas es **PageFly**. Nunca dirigir a Javier a GemPages para editar H1, FAQ o bloques de las landings.

**Cómo aplicar:** Cuando dé instrucciones de editar H1/desktop, H1/móvil, FAQ pregunta 5, schema o cualquier bloque de las landings de ciudad → decir "abre **PageFly**", no GemPages.
