# Bienvenido al Vault Churra

Este vault es el cerebro del proyecto **Churra** — gestionado por la agencia.

## Mapa de carpetas

| Carpeta | Propósito |
|---|---|
| `00-Inbox/` | Notas sin clasificar, capturas rápidas |
| `10-Clientes/Churra/` | Brief, dossier, gaps y documentos del cliente |
| `20-Pauta/` | Estrategia y ejecución de ads — Meta, Google, TikTok |
| `30-CRM-Leads/` | Análisis de funnel, pipeline, comportamiento de leads |
| `40-Contenido/` | Calendarios editoriales, guiones, copys |
| `50-Reportes/` | KPIs semanales, reportes de rendimiento |
| `90-Memoria-Claude/` | Memoria persistente de Claude — NO editar manualmente |
| `Diario/` | Notas diarias automáticas |
| `_templates/` | Plantillas para notas recurrentes |

## Cómo abrir Claude en este proyecto

```bash
cd ~/Documents/Obsidian-Churra
claude
```
